❓ Help & FAQ

Frequently Asked Questions

📌 How do I add a new item to inventory?

Go to the Inventory page, fill in all required fields (Item Name, Category, Quantity, Price, Supplier Name, Stock Status, Popular Status), and click the "Add Item" button. The system will automatically generate a unique ID for the item.

📌 How do I update an existing item?

On the Inventory page, enter the exact item name in the "Item Name" field, update the other fields as needed, and click the "Update by Name" button. Note that the item name cannot be changed.

📌 How do I delete an item?

On the Inventory page, enter the item name in the search box under "Search & Actions", then click the "Delete by Name" button. A confirmation prompt will appear to prevent accidental deletion.

📌 How do I search for items?

Go to the Search page. You can search by item name (partial matches work), and filter by category and stock status. Click "Search" to see results.

📌 What does "Popular Item" mean?

Popular items are marked with "Yes" in the Popular field. You can view only popular items by clicking the "Show Popular Items" button on the Inventory page.

📌 Is my data saved after closing the browser?

No. This application is designed for a single session only. All data will be lost when you close the browser tab or window. For persistent storage, a backend database would be required.

🔧 Troubleshooting Guide

❌ "Item with this name already exists" error

Item names must be unique. Use the "Update by Name" feature to modify an existing item instead of adding a duplicate.

❌ Item not found when updating or deleting

Make sure you are entering the exact item name (case-insensitive). Check for extra spaces or typos.

❌ Form validation errors

All required fields (marked with *) must be filled. Quantity and Price must be positive numbers. Category must be selected from the dropdown.

❌ No items appear in the list

Click the "Show All Items" button on the Inventory page. If still empty, try adding a new item using the form.

❌ Page not loading or displaying correctly

Clear your browser cache and refresh the page (Ctrl+F5 on Windows, Cmd+Shift+R on Mac). Make sure you're running ng serve in the terminal.
